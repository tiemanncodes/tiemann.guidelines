# Annotation Dimensions

This document defines the core dimensions used in guideline-based annotation.

---

## 1. Relevance

Measures how well the content satisfies the user's intent.

### Levels:
- Highly Relevant → Direct, complete answer
- Partially Relevant → Some alignment, incomplete
- Not Relevant → No meaningful connection

---

## 2. Quality

Evaluates content reliability and usefulness.

### Signals:
- Accuracy
- Clarity
- Source credibility
- Depth of information

---

## 3. Safety

Assesses potential harm.

### Categories:
- Safe
- Sensitive (context-dependent)
- Harmful

---

## 4. Intent Alignment

Checks if the result matches the type of intent:

- Informational
- Transactional
- Navigational

Mismatch reduces overall quality.

---

## 5. Trustworthiness

Evaluates whether the user can rely on the content.

### Indicators:
- Recognized sources
- Transparency
- Absence of manipulation

---

## Key Principle

High-quality annotation requires balancing all dimensions, not optimizing for a single one.
